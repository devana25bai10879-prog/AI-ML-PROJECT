Stock Market Trend Predictor

This project predicts stock market trends using machine learning techniques. It analyzes historical stock data to forecast future price movements.

 Features
- Predict stock price trends (up/down)
- Data visualization with charts
- Machine learning model training and evaluation
- Easy-to-use interface

 Tech Stack
- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib
A tool to predict the sign of stock change using Time-Lag method and Machine Learning algorithms with Keras and SK-learn, which gave a maximum accuracy of 89%(LSTM).
